

# Web UI For File Explorer Using React
## `How to Run`
- npm install
- npm start

## `Features Supported`
- Local Search
- Handle Folder and File with the same name
- Create,Delete folder and files

* Note:-`While creating a new file extention is must`

## `Technology Used`
- React.js
- CSS

## `Deployed App`
- https://react--explorer.herokuapp.com/
